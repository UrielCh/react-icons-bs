## import_map.json

For a transparent usage:

```json
{
  "imports": {
    "preact": "https://esm.sh/preact@10.15.1",
    "preact/": "https://esm.sh/preact@10.15.1/",
    "react-icons/bs": "https://deno.land/x/react_icons_bs@1.0.0//mod.ts",
  }
}
```

## Direct import sample

`import { BsFill0CircleFill } from "https://deno.land/x/react_icons_bs@1.0.0/mod.ts"`

## import_map import sample

`import { BsFill0CircleFill } from "react-icons/bs"`
